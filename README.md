Graph Library (Lab 2)

Это лабораторная работа по графам на C++.

Что реализовано:
- класс Graph (граф)
- генерация графов (полный, путь, цикл и др.)
- вычисление метрик (диаметр, мосты, двудольность и т.д.)
- парсинг графов из файлов
- сериализация в .dot и .edges
- тесты

Как запустить:

Сборка:
make

Запуск тестов:
./graph_tests

Запуск программы:
./graph_cli
